# 🏥 MediFind – Hospital Finder App

**MediFind** is a smart hospital finder and emergency support web application built using **Streamlit**. It helps users search hospitals by city and speciality, view doctor details, compare room charges, access emergency contacts, visualize hospital data through charts, and book appointments easily.

## 🚀 Features
- 🔍 Search hospitals by city and speciality  
- 👨‍⚕️ View doctor details and timings  
- 🛏️ Compare room charges  
- 📊 Interactive analytics dashboard  
- 🗺️ Hospital location maps  
- 🚨 Emergency contact system  
- 📅 Appointment booking  
- 📥 Download hospital data as CSV  

## 🛠️ Technologies Used
- Python  
- Streamlit  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Plotly  
- Folium  
- PyDeck
